본 파트에서는 전반적인 애플리케이션을 완성하여 사용자에게 전달할 수 있도록 UI 구축과 배포까지 다룹니다. RAG 기반 챗봇을 웹이나 데스크톱으로 제공하기 위한 인터페이스를 설계하고, 백엔드를 컨테이너화하여 Google Cloud의 서버리스 플랫폼으로 배포하는 과정을 실습합니다. 마지막으로 운영 중 확인해야 할 보안·비용·품질 관점의 체크리스트와 프로젝트 발표 준비 요령을 제시합니다.

**Chapter 12**에서는 Streamlit, Gradio 등 프레임워크를 이용해 간단한 사용자 인터페이스를 개발합니다. 질문 입력창, 응답 영역, 검색된 컨텍스트와 메타데이터를 함께 보여주는 레이아웃을 설계하여 투명한 챗봇을 만듭니다. 데스크톱 앱 배포를 위한 패키징 도구와 다국어 지원 팁도 다룹니다.

**Chapter 13**에서는 FastAPI/Flask 백엔드와 벡터 검색 엔진을 컨테이너 이미지로 패키징하는 방법을 배웁니다. Dockerfile 작성, requirements.txt 관리, 비밀 키와 환경 변수 주입, Cloud Build를 사용한 자동 빌드/푸시 등의 실습을 진행합니다.

**Chapter 14**에서는 컨테이너를 Google Cloud Run이나 Cloud Functions에 배포하는 과정을 설명합니다. 서비스 계정 설정과 권한 부여, 동시성 및 메모리 설정, VPC 연결, 도메인 구성, 그리고 배포 후 모니터링과 트래픽 분할까지 실습합니다.
