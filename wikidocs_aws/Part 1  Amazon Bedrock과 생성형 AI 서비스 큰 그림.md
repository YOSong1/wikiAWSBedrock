# Part 1. Amazon Bedrock과 생성형 AI 서비스 큰 그림

Part 1에서는 생성형 AI의 핵심 개념과 AWS가 제공하는 생성형 AI 서비스 전반을 다룹니다. Amazon Bedrock의 위치와 역할을 이해하고, 다른 클라우드 서비스와의 차이점을 파악합니다. 각 장에서는 실습 예제를 통해 개발을 진행할 수 있습니다.

## Part 1을 어떻게 읽어야 할 것인가?

- **Chapter 01**에서는 생성형 AI의 기본 개념, 대규모 언어 모델(LLM)과 파운데이션 모델의 이해, AWS의 생성형 AI 서비스 포트폴리오를 다룹니다.
- **Chapter 02**에서는 Amazon Bedrock의 아키텍처, 제공되는 파운데이션 모델들, 서비스 맵과 다른 AWS 서비스와의 연계 방식을 설명합니다.

각 장의 마지막에는 실습 코너가 포함되어 있습니다. 실습을 통해 예제 코드 작성과 API 호출 및 콘솔 환경 탐색을 수행하면서 학습한 내용을 확인해 보세요.
