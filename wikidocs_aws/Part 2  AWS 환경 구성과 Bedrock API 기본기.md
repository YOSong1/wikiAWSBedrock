# Part 2. AWS 환경 구성과 Bedrock API 기본기

Part 2에서는 AWS 계정과 IAM 설정, Bedrock 접근 권한 구성, SDK 기반 개발 환경 설정까지 실습합니다. Foundation Model을 API로 호출하고 Streaming 응답과 멀티모달 처리를 다룹니다. 각 장에서는 실습 예제를 통해 개발을 진행할 수 있습니다.

## Part 2를 어떻게 읽어야 할 것인가?

- **Chapter 03**에서는 AWS 계정 생성 및 보안 설정, IAM 사용자와 정책 구성, Billing 알림과 비용 추적 방법을 다룹니다.
- **Chapter 04**에서는 AWS SDK 설치, Bedrock 모델 활성화, 개발 환경 구성, 로컬과 클라우드 환경 설정을 설명합니다.
- **Chapter 05**에서는 Foundation Model API 호출 방식, 프롬프트 작성 기초, 응답 처리, Streaming과 멀티모달 입력을 다룹니다.

각 장의 마지막에는 실습 코너가 포함되어 있습니다. 실습을 통해 예제 코드 작성과 API 호출 및 콘솔 환경 탐색을 수행하면서 학습한 내용을 확인해 보세요.
