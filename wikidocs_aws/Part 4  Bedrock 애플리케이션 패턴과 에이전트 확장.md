# Part 4. Bedrock 애플리케이션 패턴과 에이전트 확장

Part 4에서는 RAG 기반 질의응답 구조를 심화하고, Agents를 활용한 업무 자동화, Guardrails를 통한 안전한 서비스 구조, 모델 평가와 Fine-tuning까지 다룹니다. 각 장에서는 실습 예제를 통해 개발을 진행할 수 있습니다.

## Part 4를 어떻게 읽어야 할 것인가?

- **Chapter 09**에서는 RAG 기반 질의응답 아키텍처, 프롬프트 엔지니어링, 응답 검증과 품질 평가를 다룹니다.
- **Chapter 10**에서는 고급 프롬프팅 기법, Few-shot Learning, Chain-of-Thought, 응답 개선 전략을 설명합니다.
- **Chapter 11**에서는 Bedrock Agents 활용, Tool 정의 및 연동, 외부 API 통합, 복잡한 업무 자동화를 다룹니다.

각 장의 마지막에는 실습 코너가 포함되어 있습니다. 실습을 통해 예제 코드 작성과 API 호출 및 콘솔 환경 탐색을 수행하면서 학습한 내용을 확인해 보세요.
