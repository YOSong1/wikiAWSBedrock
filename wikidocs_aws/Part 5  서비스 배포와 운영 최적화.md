# Part 5. 서비스 배포와 운영 최적화

Part 5에서는 구축한 AI 서비스를 운영 환경에 배포하고, 비용 최적화와 모니터링, 종합 프로젝트 설계까지 마무리합니다. 각 장에서는 실습 예제를 통해 개발을 진행할 수 있습니다.

## Part 5를 어떻게 읽어야 할 것인가?

- **Chapter 12**에서는 웹 UI 설계, 프론트엔드 프레임워크 선택, Bedrock 백엔드와의 통신 구조를 다룹니다.
- **Chapter 13**에서는 백엔드 애플리케이션 컨테이너화, Docker 이미지 빌드, AWS ECR 활용을 설명합니다.
- **Chapter 14**에서는 AWS Lambda로의 서버리스 배포, ECS/Fargate를 통한 컨테이너 오케스트레이션을 다룹니다.
- **Chapter 15**에서는 비용 최적화 전략, CloudWatch를 활용한 모니터링, 성능 튜닝, 종합 프로젝트 설계와 배포를 다룹니다.

각 장의 마지막에는 실습 코너가 포함되어 있습니다. 실습을 통해 예제 코드 작성과 API 호출 및 콘솔 환경 탐색을 수행하면서 학습한 내용을 확인해 보세요.
